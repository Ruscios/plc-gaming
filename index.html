Open Sauce · Maker & Tech Expo Build

One real PLC.
Real games, real panels.

A single AutomationDirect Productivity1000 industrial controller sits at the center of a booth where video games and hand-built Arduino control panels all speak the same language: Modbus TCP. Pull a physical throttle and an airship moves in Minecraft. Press a button and a rocket launches in Kerbal Space Program. It's all wired to the same box.

Controller P1-622 PLC Protocol Modbus TCP Server target 10.142.142.105:502 Up to 16 concurrent clients

01How it all connects

Every subproject is a Modbus client. The PLC is the one shared server in the middle. Inputs (buttons, throttles, keypads) flow in, the PLC runs the control logic, and outputs flow back out to the games — each project on its own reserved coils and registers so nothing steps on anything else.
